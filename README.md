## Configuração e instalação

Certifique-se de ter a versão mais recente do Python instalada.

```bash
git clone <repo-url>
```

```bash
pip install -r requirements.txt
```

## Executando o aplicativo

```bash
python main.py
```

## Visualizando o aplicativo

Acesse `http://127.0.0.1:5000`
